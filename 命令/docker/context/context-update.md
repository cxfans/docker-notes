# `docker context update` 命令

> 更新一个上下文

## 基本信息

### 用法

```
docker context update [OPTIONS] CONTEXT
```

### 选项

| 名称 | 描述 |
| ------- | ------- |
| `--default-stack-orchestrator` | 用于此上下文的堆栈操作的默认编排器 (swarm\|kubernetes\|all) |
| `--description` | 上下文的描述 |
| `--docker` | 设置 Docker 端点 |
| `--kubernetes` | 设置 kubernetes 端点 |
