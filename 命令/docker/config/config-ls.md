# `docker config ls` 命令

> 列出配置信息

## 基本信息

### 描述

仅 [**API 1.30**](https://docs.docker.com/engine/api/v1.30/) 及以上版本支持这个命令。

**Swarm** 这个命令可适用于 Swarm。

### 用法

```
docker config ls [OPTIONS]
```

### 选项

| 名称 | 描述 |
| --------------- | --------------------------- |
| `--filter , -f` | 根据提供的条件过滤输出 |
| `--format` | 用给定的 Go 模板格式化输出 |
| `--quiet , -q` | 只显示 IDs |
